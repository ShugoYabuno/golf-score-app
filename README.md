# Golf Score App (Balanced MVP)

## Purpose
Round中の入力速度と、ラウンド後の分析価値を両立するゴルフスコア記録アプリ。

## Tech Stack
- Next.js (App Router)
- React + TypeScript
- Tailwind CSS
- Zustand
- Dexie / IndexedDB
- Zod
- Vitest

## Core Principles
- 入力は速く、誤入力しにくく
- 分析は比較可能で、条件混在を避ける
- 機能はミニマルに（入力・履歴・分析）
